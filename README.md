# Curated GSAP Repositories

Discover a collection of sleek, minimalist GSAP repositories built with pure Vanilla JavaScript. Perfect for developers seeking clean, efficient, and dynamic animations, these curated projects showcase the power of GSAP in creating seamless user experiences.

## How to use
Unzip any of the files and ask AI to run it for you. Simple as that!

### Quick Start
1. Unzip the downloaded file.
2. Navigate into the directory:
   ```bash
   cd [unzipped-folder-name]
   ```
3. Install dependencies:
   ```bash
   npm i
   ```
4. Run the project (usually):
   ```bash
   npm run dev
   ```
